ByteBar v.2

built with JavaScript using cocktail API The Cocktail DB

The new, retro styled version of the ByteBar. This app allows users to search for cocktails by ingredient or discover new favorites by selecting the random cocktail option. Utilizing The Cocktail DB, a free and comprehensive cocktail API, ByteBar offers an extensive database of cocktail recipes and ingredients.

This project is a work in progress, continuously being enhanced visual improvements to provide a seamless and enjoyable user experience. Check out the live version here.
